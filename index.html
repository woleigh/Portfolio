  • THE FIZZBUZZ GAME I

  • The Fizz Buzz Game is an introduction to basic programming in which numbers dvisible by 3 are replaced by Fizz, numbers divisible by 5 are replaced by Buzz and numbers divisible by both 3 and 5 are replaced by FizzBuzz. It makes use of a for loop that generates the numbers based on preset high and low values

  • THE FIZZBUZZ GAME II

It makes use of a for loop that generates the numbers. The difference here is that the range is determined by the user input.

  • SHOPPING LIST APP

  • The Shopping List App was an introduction to the jquery library which enables actions/ reactions based on User input. Jquery library enabled me to add items to the shopping list and also select items for removal from the shopping list. On clicking an item, it is removed from the Shopping list.

  • HOT OR COLD GAME

  • The Hot or Cold game also involved the use of HTML, CSS and Jquery Libraries. The game aims to allow a user to guess a random number which has been chosen by the system between 1 and 100. The user guesses are then judged based on linear distance from the computer guess. The idea is that the Hot or Cold hints given to the user would to guide to the correct answer

  • STREET FIGHTER

  • One of the characters from the Street fighter game is featured in this assignment. It shows Ryu responding to instructions based on user inputs. Ryu's instructions are delivered through a combination of Jquery actions, gifs and positional images. This combination helps to simulate various postures and actions of Ryu

  • THE FLAG QUIZ

  • The Quiz game was a transition from functional Programming to Object Oriented Programming. It involves the choice between a picture of flags and the choices are listed out in an array. It is able to calculate the score obtained and start a new game if desired
